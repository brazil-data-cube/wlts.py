Version 0.8.1 (2022-07-22)

--------------------------

- Integration with the Land Cover Classification System Web Service (LCCS.py) (`#12 <https://github.com/brazil-data-cube/wlts.py/issues/12>`_).


Version 0.6.0 (2021-04-13)
--------------------------
